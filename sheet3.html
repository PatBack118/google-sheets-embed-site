Welcome to the Google Sheets Embeded Template

Need a place to display your Google Sheets but don't like sending Google's crazy long URL to people? Well here's a fairly easy solution for putting a live Google sheet on the web.

This right hand column is used for descriptive text, downloadable files (such as order forms), or links to outside resources.

I don't like this column on the right and I want it somewhere else!

Good news everyone! This column will float below when viewed on mobile due to the iframe resize function and a media query that adds a float left.

If you'd like to display your entire sheet's width and have this column below it, just change the css class .framewrap to width=100%; and the .rightCol from float=right; to float=left;
